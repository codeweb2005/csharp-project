AuthController — /api/v1/auth

Xác thực & Tài khoản

MethodEndpointAuthMô tảSource
POST/auth/loginAnonymousLogin → trả accessToken + refreshToken. PBKDF2-SHA256.Verify → GenerateAccessToken (sub email name role jti iat) + GenerateRefreshToken 64-byte. RefreshToken lưu plaintext trong DB.AuthService.LoginAsync
POST/auth/refreshAnonymousRotate token: SELECT User WHERE RefreshToken == plaintext → generate new pair. Revoke cũ.AuthService.RefreshTokenAsync
POST/auth/registerAnonymousTourist tự đăng ký: validate email/username uniqueness → PBKDF2-SHA256 hash → Role=Customer → issue JWT ngay. EmailConfirmed=false.AuthService.RegisterAsync
POST/auth/forgot-passwordAnonymousGenerate reset token 32-byte hex → SHA-256 hash stored, plaintext link gửi qua SesEmailSender hoặc NullEmailSender. No email enumeration.AuthService.ForgotPasswordAsync

Core Mobile Feature

Geofence Engine — Chi tiết kỹ thuật

📐 Thuật toán

  • Haversine formula — great-circle distance (không dùng ST_Distance)
  • Per-POI state machine: Unknown → Outside ↔ Inside
  • Debounce: DebounceCount = 3 consecutive readings trước khi fire
  • Priority resolution: nhiều POI trigger cùng lúc → OrderByDescending(Priority).First()
  • Cooldown: CooldownDuration = 10 min (configurable via SettingsPage)
  • GeofenceExited: fire ngay không debounce khi out-of-range

⚡ Events & Wiring

  • GeofenceEntered → MainViewModel.OnGeofenceEntered
  • GeofenceExited → MainViewModel.OnGeofenceExited → player.Stop()
  • Auto-play check: foreground → player.PlayAsync(audioUrl, narrationText, langCode)
  • Background → LocalNotificationService.SendGeofenceNotificationAsync(poiName)
  • Auto-play OFF → notification regardless foreground/background
  • PlaybackCompleted → SetCooldown + UpdateListenDuration + FlushVisitQueueAsync

🔊 NarrationPlayer Fallback Chain

  • 1. TryGetLocalFilePath → MediaSource.FromFile (offline ZIP audio)
  • 2. NormalizeAudioStreamUrl → thêm ?proxy=1 → MediaSource.FromUri
  • 3. OnMediaFailed → Toast → fallback TextToSpeech.SpeakAsync
  • 4. audioUrl = null → PlayTtsAsync(narrationText, langCode)
  • TTS locale: TextToSpeech.GetLocalesAsync match by langCode prefix

🗄️ VisitQueueStore

  • In-memory _pending list + Preferences backup vk_pending_visits
  • Enqueue: thêm VisitEntry (poiId langId triggerType narrationPlayed lat lng)
  • UpdateListenDuration: cập nhật listenDuration khi PlaybackCompleted
  • Dequeue: snapshot + clear
  • RestoreToQueue: trả lại nếu upload fail
  • Crash recovery: RestoreFromPreferences trong constructor

Security

Bảo mật — Kiểm tra kỹ thuật

Cơ chếImplementation thực tếFile
JWT Access TokenHMAC-SHA256. Claims: sub email name role jti iat. Không có vendorPoiId. ExpiryMinutes từ config (default 60).JwtService.GenerateAccessToken
Refresh Token64-byte random, base64. Lưu plaintext trong Users.RefreshToken. Match trực tiếp. Rotate on each use.JwtService.GenerateRefreshToken / AuthService.RefreshTokenAsync
Password HashPBKDF2-SHA256, 100,000 iterations (OWASP). Salt 16 bytes random. FixedTimeEquals.Infrastructure/Services/PasswordHasher.cs
Vendor ScopingGetVendorPOIIdsAsync: SELECT VendorPOIs WHERE VendorUserId = userId FRESH from DB. Không dùng JWT claim.BaseApiController.GetVendorPOIIdsAsync
Two-layer securityLayer 1: [Authorize] route-level. Layer 2: Service-level ownership check (poi.VendorUserId == callerId → 403).POIsController + POIService.UpdateAsync
Password Reset32-byte hex token SHA-256 hash stored. 1-hour expiry. Email via SES (prod) or Null (dev). No enumeration.AuthService.ForgotPasswordAsync
CORSAllowFrontend policy. Origins: :5173 :5174 :3000 + CORS_ALLOWED_ORIGINS env (CloudFront). AllowCredentials.Program.cs
ClockSkewClockSkew = TimeSpan.Zero — token hết hạn chính xác, không có grace period.Program.cs JwtBearer options
Mobile Token StorageSecureStorage (iOS: Keychain, Android: EncryptedSharedPreferences). Keys: vk_access_token, vk_refresh_token.ApiClient.PersistTokensAsync
