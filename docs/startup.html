Backend

ASP.NET Core 10 — Program.cs

1. AddDbContext<AppDbContext>

MySQL 8 provider (Pomelo.EntityFrameworkCore.MySql). Connection string từ appsettings.json → ConnectionStrings:DefaultConnection. Không dùng EF Migrations.

2. DI — Service Registration

JwtService đăng ký AddSingleton. Tất cả application services khác (AuthService, POIService, AudioService, DashboardService, AnalyticsService, MenuService…) đăng ký AddScoped. Repositories đăng ký AddScoped.

3. AddAuthentication JwtBearer

HMAC-SHA256. ClockSkew = TimeSpan.Zero — token hết hạn đúng giờ, không tolerance. Key từ Jwt:Key (≥ 32 ký tự).

4. AddCors

Cho phép origins từ config CORS_ALLOWED_ORIGINS (comma-separated). Dev: http://localhost:5173,http://localhost:5174.

5. AddSwaggerGen (dev only)

Tự động từ controller attributes. Chỉ enable trong app.Environment.IsDevelopment(). URL: http://localhost:5015/swagger.

6. Middleware Pipeline

UseExceptionMiddleware → UseHttpsRedirection → UseCors → UseAuthentication → UseAuthorization → MapControllers. Thứ tự này bắt buộc, không được thay đổi.

7. EnsureDefaultAdmin (Seed)

AuthService.EnsureDefaultAdmin() — create admin user nếu chưa tồn tại. Email: admin@vinhkhanh.app, password PBKDF2-SHA256(100k iter). Idempotent — chạy mỗi startup an toàn.

Ports

  • Backend API: localhost:5015 (dev)
  • Admin Panel: localhost:5173
  • Vendor Panel: localhost:5174
  • Swagger UI: localhost:5015/swagger

appsettings.json — Key Config

  • ConnectionStrings:DefaultConnection
  • Jwt:Key (≥ 32 chars), Jwt:Issuer, Jwt:Audience
  • Jwt:ExpiryMinutes (default: 60), Jwt:RefreshExpiryDays (default: 7)
  • FileStorage:Provider (local / s3)
  • AzureTTS:SubscriptionKey, AzureTTS:Region (southeastasia)

Mobile

.NET MAUI — MauiProgram.cs

Breaking change Mapsui 5.x: Registration đổi từ UseMapsuiControls() → .UseMapsui() (verified MauiProgram.cs:29). API MapInfoEventArgs đổi từ e.Feature → e.GetMapInfo(layers).

MauiProgram.cs — DI Services

  • ILocationService → AndroidLocationService / IosLocationService (platform-specific DI)
  • GeofenceEngine — Singleton, nhận ILocationService
  • NarrationPlayer — manages MediaElement + SQLite cache
  • HttpClient → configured with ApiBaseUrl from embedded appsettings.json

appsettings.json (embedded resource)

{"ApiBaseUrl": "http://10.0.2.2:5015/api/v1",
"DefaultLanguageId": 1}

10.0.2.2 = host IP từ Android Emulator. Dùng IP thật cho physical device.

GPS Permissions

  • Phase 2: LocationWhenInUse — foreground only
  • Phase 3 (planned): upgrade lên LocationAlways cho background GPS
  • Android: ACCESS_FINE_LOCATION permission trong Manifest
  • iOS: NSLocationWhenInUseUsageDescription trong Info.plist
