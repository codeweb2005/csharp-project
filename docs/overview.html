Layers

4 Layers — Clean Architecture

Layer / ProjectVai tròFiles chínhDependencies
VinhKhanh.API Presentation — HTTP, DI, JWT, Swagger Controllers.cs (all controllers), Middleware.cs, Program.cs Application + Infrastructure (DI only)
VinhKhanh.Application Use Cases — service interfaces, DTOs, validators IServices.cs (all interfaces), DTOs.cs (all DTOs), Validators/ Domain only
VinhKhanh.Domain Core — entities, enums, repository interfaces. Zero NuGet Entities.cs, Enums.cs, Interfaces.cs, BaseEntity.cs None (zero deps)
VinhKhanh.Infrastructure External — EF Core, repos, services implementation AppDbContext.cs, Repositories/POIRepository.cs, Services/{Name}Service.cs (1 file each) Domain + Application
Quy tắc bắt buộc: VinhKhanh.Domain KHÔNG được thêm bất kỳ NuGet package nào. File .csproj của Domain chỉ chứa <TargetFramework>net10.0</TargetFramework>. Vi phạm quy tắc này sẽ làm hỏng kiến trúc.

Database

MySQL 8 — Database Schema

Database: vinhkhanh. Engine: InnoDB. Collation: utf8mb4_unicode_ci. Quản lý schema bằng numbered SQL scripts (Database/00N_*.sql) — không dùng EF Core Migrations.

TableMô tảKey Fields
POIsĐiểm ẩm thực — dữ liệu gốcLatitude, Longitude, GeofenceRadius, Priority, VendorUserId, TotalVisits
POITranslations1 row per POI per Language — tên, mô tả, NarrationTextPOIId FK, LanguageId FK, Name, NarrationText, Highlights(JSON)
UsersAdmin / Vendor / Customer — unified tableEmail, PasswordHash (PBKDF2-SHA256, 100k iter), Role (enum), IsActive
AudioNarrationsMP3 files per POI per LanguagePOIId, LanguageId, FilePath (S3 or local), DurationSeconds, TTSGenerated
VisitHistoryGeofence trigger events — analytics + auditPOIId, UserId(nullable), TriggerType(enum), NarrationPlayed, ListenDuration
CategoriesPhân loại POI (Bún Bò, Phở, Bánh Mì…)Icon, Color, IsActive, SortOrder
LanguagesNgôn ngữ được hỗ trợCode (vi/en/zh…), NativeName, FlagEmoji, IsActive
MenuItemsThực đơn của từng quánPOIId, Name, Price, Category, IsSignature, ImagePath
RefreshTokensJWT refresh token store (SHA-256 hash)UserId, Token(hashed), ExpiresAt, IsRevoked

📍 Spatial Index

ST_Distance_Sphere(POINT(lng, lat), POINT(poi.Longitude, poi.Latitude)) — POINT nhận (longitude, latitude), không phải (lat, lng). Dùng cho GET /pois/nearby.

🔢 TotalVisits Denormalized

Trường TotalVisits trên bảng POIs là denormalized — tăng khi có VisitHistory mới, không dùng COUNT(*). Tránh slow query trên bảng VisitHistory lớn.

Roles

3 User Roles

Role (enum)Mô tảTruy cập APIJWT claim
Admin Quản trị viên hệ thống Toàn bộ — POIs, Users, Settings, Analytics, Offline Packages role=Admin
Vendor Chủ quán Chỉ POI của mình — listing, menu, audio, analytics riêng role=Vendor + vendorPoiId
Customer Du khách (mobile app) Public endpoints (AllowAnonymous) — view POI, audio stream, sync role=Customer (optional, anonymous OK)
