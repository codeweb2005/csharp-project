🌐 Đa Ngôn Ngữ — POI Translations

Mỗi POI có bảng POITranslations — 1 row per language. GET /languages public endpoint. langId param lọc translation. Azure TTS gen per language.

i18n Model

POITranslations — Data Model

Ngôn ngữ không được hardcode — mọi thứ dựa trên bảng Languages. Mỗi kết hợp POI + Language = 1 row trong POITranslations.

Languages Table

IdCodeNativeNameFlag
1viTiếng Việt🇻🇳
2enEnglish🇬🇧

GET /api/v1/languages — Anonymous. Dùng bởi mobile app khi hiển thị language picker.

POITranslations Fields

  • POIId — FK → POIs.Id
  • LanguageId — FK → Languages.Id
  • Name — Tên quán bằng ngôn ngữ tương ứng
  • ShortDescription — Mô tả ngắn (hiển thị trên map popup)
  • FullDescription — Mô tả đầy đủ (detail page)
  • NarrationText — Script cho Azure TTS đọc khi geofence trigger
  • Highlights — JSON array (e.g. ["Bún bò đặc biệt", "Giá bình dân"])
Highlights serialization: Highlights là JSON string trong MySQL — serialize/deserialize bằng System.Text.Json.JsonSerializer. Không dùng thư viện ngoài.

API Usage

langId Param — Filter Translation

EndpointlangId paramHành vi
GET /api/v1/pois/{id}/public?langId=2Return translation của language 2 (en). Nếu không có → trả vi (fallback)
GET /api/v1/pois/nearby?langId=1JOIN POITranslations WHERE LanguageId=1 → name, shortDescription trên map
GET /api/v1/sync/delta?languageId=1Delta sync — trả translations của language đó cho mobile cache
