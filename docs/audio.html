Azure TTS

Azure Cognitive Services TTS

Cấu hình

  • Service: Azure Cognitive Services — Speech
  • Region: southeastasia
  • Config keys: AzureTTS:SubscriptionKey, AzureTTS:Region
  • Format output: MP3 audio stream
  • API: Speech Synthesis (SSML) REST endpoint

Voice — vi-VN-HoaiMyNeural

  • Tiếng Việt: vi-VN-HoaiMyNeural (Female, chất lượng tốt nhất cho Vietnamese)
  • Admin nhập voiceName vào request — có thể dùng voice khác
  • TTS chỉ chạy khi NarrationText đã được điền trong POITranslation
  • Sau khi gen: lưu MP3 vào storage, INSERT AudioNarrations với TTSGenerated=true

File Storage

  • FileStorage:Provider = local → lưu wwwroot/audio/ (dev)
  • FileStorage:Provider = s3 → PUT lên FileStorage:S3BucketName
  • DI conditional registration trong Program.cs
  • Filename pattern: poi-{id}-lang-{id}-{uuid}.mp3

AudioNarrations Table

  • IsDefault — audio mặc định cho ngôn ngữ đó
  • IsActive — soft disable không xóa file
  • TTSGenerated — phân biệt upload vs auto-gen
  • DurationSeconds — thời lượng audio (tính khi gen)
Vendor audio access: Vendor có thể upload MP3 và generate TTS cho POI của mình. AuthService kiểm tra poi.VendorUserId == callerId trước khi cho phép. Admin có thể thao tác trên mọi POI.
