Luong 1 — Admin Ket Noi SignalR

useMonitorHub connects to TourMonitorHub via SignalR WebSocket

Entry point: LiveMonitorPage mount → useMonitorHub({ onEnter, onExit, onMove, onWebVisitorUpdate }) → useEffect([enabled, connect]) fires → connect() async → connection.start() → connection.invoke("JoinMonitorGroup").
Đồng thời: fetchAll() gọi song song presence.getStats() và presence.getSnapshot() mỗi SNAPSHOT_INTERVAL_MS = 10_000 ms.

Luong 2 — Tourist App + Visitor Website Kick Hoat Broadcast

POST /presence/update (Tourist JWT) & POST /presence/web-visitor* (Anonymous)

Verified from Controllers.cs + PresenceService.cs: Tourist dùng POST /presence/update [Tourist JWT] — switch(req.EventType) gọi 3 method khác nhau. Web visitor (visitor site) dùng 3 endpoint riêng [AllowAnonymous]: POST /presence/web-visitor (heartbeat), POST /presence/web-visitor/exit (rời trang), POST /presence/web-location (GPS). Web visitor count được đếm từ WebVisitorLastSeen (ConcurrentDictionary in-memory, stale 2 min).

Luong 3 — REST Polling (fetchAll) + GetSnapshotAsync + GetDashboardStatsAsync + PresenceCleanupService

fetchAll() every 10s: getStats() + getSnapshot() song song — PurgeStaleAsync moi 5 phut

REST Polling: fetchAll() trong LiveMonitorPage dùng Promise.all gọi song song presence.getStats() → GET /presence/stats [Admin] và presence.getSnapshot() → GET /presence/snapshot [Admin], mỗi SNAPSHOT_INTERVAL_MS = 10_000 ms và ngay sau mỗi SignalR event.
Cleanup: PresenceCleanupService (BackgroundService) gọi PurgeStaleAsync(TimeSpan.FromMinutes(15)) mỗi 5 phút.
