Luong 1 — Admin Ket Noi SignalR

useMonitorHub connects to TourMonitorHub via SignalR WebSocket

Entry point: LiveMonitorPage renders → useMonitorHub({ onEnter, onExit, onMove }) → useEffect fires → connect() async → connection.start() → connection.invoke("JoinMonitorGroup").

Luong 2 — Tourist Kich Hoat Broadcast Real-time

POST /api/v1/presence/update (single endpoint, EventType dispatch)

IMPORTANT — verified from Controllers.cs: There is only one endpoint: POST /api/v1/presence/update [Authorize(Roles = "Tourist")]. The field req.EventType drives a switch to call different PresenceService methods. sessionId is extracted from the JWT claim (User.FindFirst("sessionId")?.Value) — it is NOT in the request body.
sequenceDiagram autonumber actor Tourist as Tourist App (Role=Tourist JWT) participant CTRL as PresenceController.Update() participant SVC as PresenceService.cs participant DB as ActivePresence (MySQL) participant HUB as IHubContext TourMonitorHub participant GRP as Group Admins (SignalR) participant UMH as useMonitorHub.js participant LMP as LiveMonitorPage.jsx Tourist->>CTRL: POST /api/v1/presence/update
Header: Authorization Bearer Tourist-JWT
Body: PresenceUpdateRequest { EventType, PoiId?, Latitude, Longitude } activate CTRL Note over CTRL: sessionId = User.FindFirst("sessionId")?.Value ?? ""

.OrderBy(LanguageId).Select(Name).FirstOrDefaultAsync() DB-->>SVC: poiName SVC->>HUB: hub.Clients.Group("Admins").SendAsync("TouristEnteredPOI",
{ eventType="enter", sessionId[..8]+"...", poiId, poiName, lat, lng, timestamp=UtcNow }) HUB->>GRP: Broadcast TouristEnteredPOI to all Admins GRP->>UMH: connection.on("TouristEnteredPOI") fires UMH->>LMP: onEnter(msg) -> handleEnter(msg) LMP->>LMP: setConnected(true) LMP->>LMP: pushEvent({ ...msg, eventType: "enter" }) LMP->>LMP: fetchSnapshot() deactivate SVC else EventType.ToLower() == "exit" Note over CTRL,LMP: case "exit" CTRL->>SVC: TrackExitAsync(sessionId, req.PoiId) activate SVC SVC->>DB: ActivePresence.FindAsync(sessionId) DB-->>SVC: presence row (no-op if null) SVC->>DB: presence.PoiId=null; presence.UpdatedAt=UtcNow; SaveChangesAsync() DB-->>SVC: OK Note over SVC: Only broadcasts if req.PoiId.HasValue SVC->>HUB: hub.Clients.Group("Admins").SendAsync("TouristExitedPOI",

Luong 3 — REST Polling + PresenceCleanupService

fetchSnapshot() every 10s + PurgeStaleAsync every 5min

REST Polling: LiveMonitorPage calls presence.getSnapshot() → GET /api/v1/presence/snapshot [Admin] every 10 000 ms (SNAPSHOT_INTERVAL_MS = 10_000). Also called immediately after each SignalR event via pushEvent(). Cleanup: PresenceCleanupService (BackgroundService) calls PurgeStaleAsync(TimeSpan.FromMinutes(15)) every 5 minutes.
