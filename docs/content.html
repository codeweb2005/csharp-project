POI REST Endpoints

EndpointAuthMô tả
GET /api/v1/poisAdmin / VendorPaginated list. Vendor → tự động scoped về vendorPoiId JWT claim. Params: page, size, search, categoryId, isActive, sortBy, order
GET /api/v1/pois/{id}Admin / VendorFull detail kèm translations[] array. Vendor → 403 nếu không phải POI của họ
GET /api/v1/pois/{id}/publicAnonymous ✅POI detail cho mobile app (chỉ active). Query: ?langId=1 lọc translation
GET /api/v1/pois/nearbyAnonymous ✅Spatial query với ST_Distance_Sphere. Query: lat, lng, radius (max 5000m), langId
POST /api/v1/poisAdmin onlyCreate POI + translations (all languages in one request). FluentValidation.
PUT /api/v1/pois/{id}Admin / VendorUpdate. Vendor → 403 nếu poi.VendorUserId != callerId
DELETE /api/v1/pois/{id}Admin onlyDelete POI + cascade AudioNarrations, POITranslations, MenuItems
PATCH /api/v1/pois/{id}/toggleAdmin / VendorToggle IsActive
PATCH /api/v1/pois/{id}/featuredAdmin onlyToggle IsFeatured

{ "categoryId": 2, "address": "123 Vĩnh Khánh, P8, Q4, TP.HCM", "latitude": 10.7553, "longitude": 106.7017, "geofenceRadius": 50, // metres — tùy chỉnh per POI "priority": 1, // higher = trigger first trong overlap "phone": "028-1234-5678", "priceRangeMin": 30000, "priceRangeMax": 80000, "openingHours": "06:00 – 22:00", "vendorUserId": 5, // null = Admin-managed "translations": [ { "languageId": 1, // 1=vi, 2=en, … "name": "Quán Bún Bò Cô Linh", "shortDescription": "Bún bò nổi tiếng Q4", "fullDescription": "...", "narrationText": "Chào mừng bạn đến...", // dùng cho Azure TTS "highlights": ["Bún bò đặc biệt", "Giá bình dân"] // JSON array } ] }
Response Envelope: Mọi response đều wrap trong {"success": true, "data": {...}, "error": null}. Admin dùng ApiResult() helper từ BaseApiController để map error code → HTTP status. Error codes: NOT_FOUND→404, FORBIDDEN→403, UNAUTHORIZED→401, VALIDATION_ERROR→400.

Nearby Query — ST_Distance_Sphere

SELECT pois.*, t.Name, t.ShortDescription, ST_Distance_Sphere( POINT(:lng, :lat), -- POINT(longitude, latitude) ! POINT(poi.Longitude, poi.Latitude) ) AS Distance FROM POIs poi JOIN POITranslations t ON t.POIId = poi.Id AND t.LanguageId = :langId WHERE poi.IsActive = 1 AND ST_Distance_Sphere( POINT(:lng, :lat), POINT(poi.Longitude, poi.Latitude) ) <= :radius ORDER BY Distance ASC
⚠️ POINT(longitude, latitude) — không phải (lat, lng). MySQL ST_Distance_Sphere nhận theo thứ tự (longitude, latitude). Đây là source of bug phổ biến.
