Tổng Quan Dự Án

🎯 Mục tiêu

Audio tour tự động cho du khách đến phố ẩm thực Vinh Khanh. Khi bước vào vùng geofence của quán, app MAUI phát thuyết minh MP3 (tiếng Việt, tiếng Anh…) về quán đó — không cần thao tác.

👥 3 User Roles

  • Admin — quản lý toàn hệ thống, CRUD POI/User/Category
  • Vendor — chủ quán, chỉ quản lý POI của mình (JWT-scoped)
  • Customer — du khách dùng mobile app, anonymous hoặc register

🔐 Security Model

  • Bearer JWT (HMAC-SHA256), ClockSkew=Zero
  • PBKDF2-SHA256 100,000 iterations (OWASP)
  • Refresh token rotation + SHA-256 hash lưu DB
  • Vendor scoping qua vendorPoiId JWT claim

📦 Cấu trúc codebase

  • backend/src/ — 4 projects (API/App/Domain/Infra)
  • admin-frontend/ — port 5173
  • vendor-frontend/ — port 5174
  • mobile/VinhKhanh.Mobile/ — .NET MAUI
  • Database/001–009_*.sql — raw SQL migrations
Convention đặc biệt: Dự án dùng pattern "1 file cho mỗi loại" — toàn bộ controllers trong Controllers.cs, DTOs trong DTOs.cs, interfaces trong IServices.cs. Chỉ service implementations trong Infrastructure/Services/ mới có file riêng. Không dùng EF Migrations — chỉ raw SQL scripts.
