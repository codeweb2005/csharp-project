Convention

One-File Convention

Dự án dùng pattern "1 file cho mỗi loại" để dễ nhìn toàn bộ API surface trong 1 lần. Chỉ service implementations trong Infrastructure mới có file riêng.

LoạiFile tập trungCách thêm mới
All ControllersAPI/Controllers/Controllers.csThêm controller mới vào CUỐI file, ngăn cách bằng // ===========
All MiddlewareAPI/Middleware/Middleware.csTương tự Controllers
All Service InterfacesApplication/Services/IServices.csThêm interface mới vào cuối file
All DTOsApplication/DTOs/DTOs.csThêm DTO/Request/Response mới vào cuối file
All EntitiesDomain/Entities/Entities.csThêm entity mới vào cuối file
All EnumsDomain/Enums/Enums.csThêm enum mới vào cuối file
Domain InterfacesDomain/Interfaces/Interfaces.csRepository interfaces
Service ImplementationsInfrastructure/Services/{Name}Service.cs1 file riêng per service (ngoại lệ duy nhất)
KHÔNG tạo file riêng cho từng controller, DTO, service interface, entity, hoặc enum. Chỉ service implementations trong Infrastructure được tách file. Vi phạm quy tắc này sẽ bị reject khi code review.

API Response

ApiResponse<T> Envelope

Success Response

{ "success": true, "data": { /* payload */ }, "error": null }

Error Response

{ "success": false, "data": null, "error": { "code": "NOT_FOUND", "message": "POI not found" } }
Error CodeHTTP StatusKhi nào
NOT_FOUND404Resource không tồn tại
FORBIDDEN403JWT valid nhưng không đủ quyền (ví dụ: Vendor truy cập POI của người khác)
UNAUTHORIZED401Thiếu hoặc expired JWT
VALIDATION_ERROR400FluentValidation fail
any other400General client error

DI Patterns

Dependency Injection Scoping

ServiceLifetimeLý do
JwtServiceSingletonStateless, expensive init — chia sẻ toàn app
AuthService, POIService, AudioService, …ScopedDefault. 1 instance per HTTP request → đúng với DB context lifecycle
IRepository<T>ScopedGeneric repository — tied to AppDbContext scope
IPOIRepositoryScopedSpecialized repo với custom spatial queries
FileStorage (local/s3)ScopedDI conditional dựa trên FileStorage:Provider config
Thêm service mới — checklist 3 bước: (1) Thêm interface vào Application/Services/IServices.cs → (2) Tạo implementation Infrastructure/Services/{Name}Service.cs → (3) Đăng ký AddScoped<IXxxService, XxxService>() trong Program.cs section "Application Services".
