📡 GPS Geofence — Tự Động Phát Audio

MAUI GeofenceEngine nhận GPS updates từ ILocationService, tính khoảng cách Haversine, trigger NarrationPlayer khi vào zone POI.

Core Flow

Geofence Processing Loop

GPS Update (ILocationService) → Throttle 5s check → Load POIs from SQLite → Haversine Distance → Cooldown check → Trigger Narration → Queue VisitHistory

ILocationService (Platform)

  • Android: AndroidLocationService — FusedLocationProvider
  • iOS: IosLocationService — CLLocationManager
  • Accuracy: Best available (GPS)
  • Throttle: báo lên GeofenceEngine mỗi 5 giây (tránh battery drain)
  • Phase 3: nâng lên LocationAlways cho background mode

GeofenceEngine

  • Singleton — tồn tại suốt app lifetime
  • Haversine distance tính bằng C# thuần (không cần thư viện ngoài)
  • GeofenceRadius mỗi POI khác nhau (field trên POIs table)
  • Priority field: khi 2 POI overlap → POI Priority cao hơn trigger trước
  • Cooldown per POI ID — tránh phát narration lặp lại khi đứng yên

NarrationPlayer

  • Kiểm tra SQLite cache trước → play local MP3 (0ms latency)
  • Nếu không có → GET /api/v1/audio/{id}/stream (AllowAnonymous endpoint)
  • Stream về → cache vào SQLite để offline lần sau
  • TriggerType: GeofenceEnter / ManualPlay / Offline
  • Phase 3: local notification khi trigger ở background
